..sub_title..
Install SSH Over LAN
..code..
sudo apt install openssh-server
sudo systemctl status ssh
# press >> q << to get back


..sub_title..
Enable & Disable SSH Over LAN
..code..
# Disabling SSH on Ubuntu
sudo systemctl disable ssh
# Enable SSH on Ubuntu
sudo systemctl enable ssh


..text..
